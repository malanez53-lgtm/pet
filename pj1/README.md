# Проект: Главная страница «Киберспорт"

Статическая страница, воссоздающая макет главной страницы из примера (без центрального изображения — вместо него декоративный светящийся круг).

Как открыть локально:

1. Откройте терминал (zsh) и перейдите в папку проекта:

```bash
cd /Users/avroraivanova/Desktop/pj1
```

2. Откройте `index.html` в браузере (macOS):

```bash
open index.html
```

Или просто двойной клик по `index.html` в Finder.

Дальше (опции):
- Подставить реальную картинку вместо декоративного круга, заменив блок `.glow-wrap` содержимым `<img>`.
- Добавить адаптивное меню и анимации кнопок.
